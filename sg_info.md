### Project 2: everBrain - Secure Password Manager

**Subtitle:** EverBrain is a secure password management mobile app built with Flutter and other techonology
**Content:** EverBrain is a secure password management mobile application that allows users to conveniently store passwords for various online accounts. It utilizes advanced encryption algorithms to securely store passwords locally on the user's device. Key features include a robust password generator, integration with Huawei Mobile Services for additional security, and strict identity verification requiring a master passcode or biometrics each time the app is accessed. The app aims to significantly enhance password security while also simplifying password management across multiple platforms for users.
